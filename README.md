# Ecommerce_data_analysis-
# Data Analytics Project using Python

# This project focuses on data analysis using Python libraries such as
NumPy, Pandas, Matplotlib, and Seaborn. The goal is to extract insights,
clean data, and visualize trends using real-world datasets.

## Tools & Libraries :
- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Seaborn

## Dataset
The dataset contains information related to ECOMERCE.
It was cleaned and processed before analysis.

## Analysis Performed
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Statistical Summary
- Data Visualization
- Insight Generation

## Visualizations
- Bar charts
- Line graphs
- Heatmaps
- Distribution plots

## Project Report
A detailed project report is available in the ( report file )


## How to Run
1. Clone the repository
2. Install required libraries
3. Open Jupyter Notebook
4. Run Python_sql_ecomerce.ipynb
